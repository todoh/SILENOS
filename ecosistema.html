El Ecosistema

Silenos es un entorno unificado. Una arquitectura donde las herramientas narrativas, la orquestación de IA y el almacenamiento local se fusionan en un solo motor.

Silenos V4 ↗
Fase 01 // Fundamentos

Koreh Suite

Nuestra herramienta propietaria. Un motor narrativo que actúa como arquitecto y auditor estructural, permitiendo la creación de mundos literarios coherentes, bases de datos de lore masivas y asistencia en la escritura sin bucles iterativos.

Silenos Films

Productora audiovisual del ecosistema. Películas, cortometrajes y animaciones generadas y producidas utilizando la tecnología de Silenos y herramientas de IA.

Fase 02 // Entorno V4

El núcleo de todo. Un entorno de escritorio minimalista que opera sobre tu disco duro local y alberga nuestra orquestación de IA.

Animation Studio

Plataforma de producción de escenas. Convierte guiones en composiciones visuales completas. Orquesta la generación paralela de fondos y sprites calculando automáticamente la perspectiva y el layout.

Generador Libros

OmniGen v5.3. Sistema de tres agentes (Arquitecto, Escritor, Crítico) para escribir novelas coherentes sin bucles repetitivos usando 'Memoria Larga'.

Datos Studio

Fábrica de entidades masiva. Integra 'Chroma Core' para procesar imágenes generadas, eliminar fondos verdes (Despill) y crear assets transparentes listos para animación.

Cronología

Arquitecto temporal. Visualiza la trama en una línea de tiempo infinita e incluye un 'Auditor de Densidad' que calcula los capítulos necesarios basados en los beats narrativos.

Fase 03 // Pollination

Nuestra interfaz unificada de inferencia. Conexión sin fricciones con modelos masivos de IA generativa para orquestar texto, imagen, video 3D y audio en un solo puente de datos integrado.

Audio & Video

Generación de secuencias audiovisuales y pistas de música directas desde el prompt mediante orquestación de modelos externos.

Render 3D & SVG

Creación paramétrica de modelos tridimensionales visualizables en el entorno local y síntesis de vectores escalables (SVG).

Storyboard Adv

Planificación visual secuencial. El motor desgrana guiones y proyecta encuadres mediante un flujo de trabajo optimizado.

Fase 04 // Arquitectura Técnica

App Bridge & LocalFS

Silenos opera directamente sobre el disco duro del usuario utilizando la File System Access API nativa. No existe base de datos en la nube para los archivos del usuario, garantizando soberanía total, privacidad absoluta y persistencia real en tiempo real.

Universe Engine

El núcleo visual del sistema de archivos. Sustituye las convencionales listas de carpetas por un espacio físico interactivo renderizado en HTML5 Canvas, incorporando gravedad, colisiones y funciones de zoom infinito para organizar datos espaciales.

Window Manager

Sistema avanzado de gestión de ventanas flotantes. Soporta multitarea asíncrona real, redimensionado libre, anclaje (docking) y persistencia del estado visual. Implementa una rigurosa gestión de foco y z-index.

Estructura Lore

Toda la narrativa generada por OmniGen y los datos creados en el Datos Studio se guardan en estructuras de array JSON estrictas. Esto permite que herramientas externas consuman la base de datos de manera procedural sin errores.
