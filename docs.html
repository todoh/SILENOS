DOCs.

SYSTEM.CORE

APP BRIDGE & LOCALFS

Silenos opera directamente sobre el disco duro del usuario utilizando la File System Access API nativa de los navegadores modernos. No existe base de datos en la nube para los archivos del usuario, garantizando soberanía total, privacidad absoluta y persistencia real en tiempo real.

SYSTEM.UI

UNIVERSE ENGINE

El núcleo visual del sistema de archivos. Sustituye las convencionales listas de carpetas por un espacio físico interactivo renderizado en HTML5 Canvas, incorporando gravedad, colisiones y funciones de zoom infinito para organizar datos espaciales.

SYSTEM.UX

WINDOW MANAGER

Sistema avanzado de gestión de ventanas flotantes. Soporta multitarea asíncrona real, redimensionado libre, anclaje (docking) y persistencia del estado visual. Implementa una rigurosa gestión de foco y z-index.

SYSTEM.DATA

ESTRUCTURA LORE

Toda la narrativa generada por OmniGen y los datos creados en el Datos Studio se guardan en estructuras de array JSON estrictas. Esto permite que herramientas externas, como Nexus o el juego de cartas, consuman la base de datos de manera procedural sin errores.
