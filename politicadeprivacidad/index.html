POLÍTICA DE PRIVACIDAD

Última actualización: Julio de 2026

1. TITULARIDAD DEL SITIO Y LA APLICACIÓN

La aplicación "Silenos Libros" es operada como una plataforma de acceso a manuscritos y contenido literario de acceso público o privado, conectando directamente con repositorios remotos sin intermediarios.

2. DATOS RECOGIDOS Y ALMACENAMIENTO

Nuestra aplicación no recopila, transmite ni almacena ningún tipo de información de identificación personal (PII) en servidores externos ajenos a la infraestructura de ejecución. Todo el progreso de lectura, marcas de favoritos y configuraciones de visualización se guardan estrictamente de forma local en el dispositivo del usuario utilizando las capacidades de almacenamiento web nativas (LocalStorage).

3. PERMISOS REQUERIDOS

La aplicación requiere acceso a Internet para la sincronización asíncrona de los archivos de lectura desde el repositorio público configurado y para la correcta carga de recursos tipográficos y matemáticos.

4. SERVICIOS DE TERCEROS

El contenido se indexa dinámicamente utilizando la API pública de GitHub. Dicho servicio de terceros se rige exclusivamente por sus propias políticas de privacidad y condiciones de servicio.

5. CONTACTO

Para cualquier duda respecto a esta declaración de transparencia, puede contactar a través del ecosistema global en silenos.es.
